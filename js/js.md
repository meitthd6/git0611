### 데이터/변수/연산자

- 데이터 
  - 숫자
  - 문자
```
숫자 : 0,1,2,3,4,5...
문자 : 가,나,다... , a,b,c...

```
- 데이터 타입(종류)
  - 숫자 : 정수, 실수 ...
  - 문자 
    - 코드상에서 문자는 따옴표로 묵임
    - 낱개 글자, 묶음 글자 ...
```

1 + 1 = 2
'a' + 1 = 'a1'
'1' + 1 = 11

```

** 자바스크립트는 데이터 타입을 엄격하게 구분하지 않음
